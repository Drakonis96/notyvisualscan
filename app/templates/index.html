    Welcome to NotyVisualScan

    NotyVisualScan is a tool that automates various processes in Notion using artificial intelligence. It offers the following functionalities:

    • Description üöÄ: Automatically generate detailed descriptions of images. (Process)
    • Tagging üè∑Ô∏è: Analyze descriptions and assign tags based on a given prompt. (Process)
    • Comparator ‚öñÔ∏è: Compare entries and determine an outcome based on context. (Process)
    • AI Comparator üöÄ: Use AI to analyze contexts and generate outputs. (Process)
    • File Upload üìÇ: Upload files and automatically attach them to Notion database rows by matching file names with a context column. Supports Exact and Approximate matching modes.
      • Suggested file/context name format: BD001-FG001
      • Exact match: The file name (without extension) must be exactly the same as the context value (e.g., BD001-FG001 matches only BD001-FG001).
      • Approximate match: The part before the dash must be identical, and the part after the dash is compared ignoring leading zeros in the numeric part (e.g., BD001-FG000001 will match BD001-FG1 or BD001-FG001).
    • Automation üîÄ: Chain multiple processes (Description, Tagging, Comparator, etc.) into a single automated workflow. You can save and load your favorite automation step sequences for quick reuse. (Process)
    • Notifications üîî: Receive real-time Pushover notifications when a process or automation starts or ends. Configure which events trigger notifications in the Notifications section. (Config)

    Before starting any process, please ensure you have properly configured the required settings in the corresponding configuration sections and in General Configs (DB, Models, Languages, Prompts, and Backup).

    Recommendations
    • Column names in Notion are case-sensitive. Always use the exact spelling, including uppercase and lowercase letters.
    • Do not use spaces or special characters in column names to avoid possible errors.
    • For file uploads, follow the suggested naming format and choose the matching mode that best fits your use case.

    Note: File Upload cannot be automated in Automation workflows at this time.
